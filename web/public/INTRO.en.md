# nof — Multi-Trading Type / Multi-Model AI Trading Experiment System (Simplified)

nof is an open-source AI trading system for research and rapid experimentation:

- Supports multiple trading types: Futures test (demo-futures), Spot test (test-spot), local simulation, and live trading.
- Supports multiple models: Qwen / DeepSeek / GLM, etc. Switch by changing API Key or preset.
- Prompt Studio: Visual management of system/user templates, independently managed by environment or Bot, with hot-reload.
- Multi-Bot parallel: Each with independent account status, conversations, trades, with chart/leaderboard comparison at a glance.

For quick start and complete documentation, please see README.md in the project root.
