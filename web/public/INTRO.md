# nof — 多交易类型 / 多模型的 AI 交易实验系统（简版）

nof 是一个面向研究与快速试验的开源 AI 交易系统：

- 支持多交易类型：期货测试（demo-futures）、现货测试（test-spot）、本地模拟，以及实盘。
- 支持多模型：Qwen / DeepSeek / GLM 等，通过更换 Key 或预设即可切换。
- Prompt Studio：可视化管理系统/用户模板，按环境或 Bot 独立管理，热更新生效。
- 多 Bot 并行：各自独立的账户状态、会话、成交，图表/排行榜对比一目了然。

快速开始与完整说明请查看项目根目录 README.md。
